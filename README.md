# Fundación Creemos Región — Sitio web

Sitio web estático (HTML/CSS/JS) de la **Fundación Creemos Región**.

## Páginas

- `index.html`: página principal.
- `transparencia-esal.html`: repositorio público de documentos (ESAL).

## Documentos (carpeta `docs/`)

Actualmente publicados para descarga en `transparencia-esal.html`:

- `docs/estados-financieros/ESTADOS FINANCIEROS 2025.pdf`
- `docs/revisoria-fiscal/DICTAMEN REVISOR 2026.pdf`

### Dónde poner nuevos documentos

- Estados financieros: `docs/estados-financieros/`
- Revisoría fiscal (dictámenes, informes del revisor, etc.): `docs/revisoria-fiscal/`
- Otros (si se activan más secciones): `docs/estatutos/`, `docs/certificaciones/`, `docs/informes/`

Si cambia nombres o rutas, actualice los enlaces en `transparencia-esal.html`.
